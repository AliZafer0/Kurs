Pseudo-Elements Showcase

Başlık Altı Çizgisi

Bu başlık, altına bir çizgi eklemek için `::before` pseudo-elementi kullanıyor.


h2::before {
    content: ""; /* Pseudo-element ile boş bir içerik ekler */
    position: absolute; /* Çizgiyi kesin konumlandırma ile başlığın altında yerleştirir */
    bottom: -10px; /* Çizgiyi başlığın 10px altına yerleştirir */
    left: 50%; /* Çizgiyi yatayda ortalar */
    transform: translateX(-50%); /* Yatayda ortalanan çizgiyi başlığın tam ortasına getirir */
    width: 50px; /* Çizginin genişliğini 50px olarak ayarlar */
    height: 5px; /* Çizginin yüksekliğini 5px olarak ayarlar */
    background-color: #3498db; /* Çizginin rengini mavi yapar */
    border-radius: 2px; /* Çizginin köşelerini yuvarlatır */
}

Bu kod, başlık altına mavi bir çizgi ekler. Çizgi `::before` pseudo-elementi ile oluşturulur ve başlığın altında, ortalanmış olarak yerleştirilir.

}

Liste öğelerinin önüne onay işareti eklemek için `::before` kullanılır. Bu, öğelerin yanına ikonlar eklemek için idealdir.

Alıntı Çizgisi ve İkonu

Alıntılar genellikle daha dikkat çekici olmak için stillendirilir.

Bu bir alıntıdır. Pseudo-elementler ile stillendirildi. 

blockquote::before {
    content: "“"; /* Alıntının başına büyük bir tırnak işareti ekler */
    font-size: 3em; /* Tırnak işaretinin boyutunu ayarlar */
    color: #3498db; /* Tırnak işaretinin rengini mavi yapar */
    position: absolute; /* Tırnak işaretini kesin konumlandırır */
    left: 10px; /* Tırnak işaretini sol tarafa yerleştirir */
    top: -10px; /* Tırnak işaretini alıntının biraz üstüne yerleştirir */
}

Bu stil, alıntıların yanına büyük bir tırnak işareti ekler, bu da alıntının vurgulanmasını sağlar.

Buton Süslemeleri

Butonların başına ve sonuna ikonlar ekleyerek görsel olarak zenginleştirmek için `::before` ve `::after` kullanımı:

Daha Fazla Bilgi 

.button::before {
    content: "🚀"; /* Butonun başına bir roket ikonu ekler */
    margin-right: 8px; /* Buton metni ile ikon arasında boşluk bırakır */
    display: inline-block; /* İkonun buton içinde düzgün bir şekilde görüntülenmesini sağlar */
}
.button::after {
    content: "➡"; /* Butonun sonuna bir ok ikonu ekler */
    margin-left: 8px; /* Buton metni ile ikon arasında boşluk bırakır */
    display: inline-block; /* İkonun buton içinde düzgün bir şekilde görüntülenmesini sağlar */
}

Butonlara ön ve arka ikonu ekleyerek, butonların daha dinamik ve etkileşimli görünmesini sağlar.

Sosyal Medya İkonları

Sosyal medya ikonlarının arka plan rengini ve hover efektlerini `::before` ile nasıl stilize edebileceğinizi gösteren bir örnek:

F T G+ L 

.social-icon::before {
    content: ""; /* İkonun arka planını oluşturmak için boş içerik ekler */
    position: absolute; /* Arka planı ikonun tam arkasına yerleştirir */
    top: 0; /* Arka planı ikonun üstüne hizalar */
    left: 0; /* Arka planı ikonun soluna hizalar */
    width: 100%; /* Arka planın genişliğini ikonla eşleştirir */
    height: 100%; /* Arka planın yüksekliğini ikonla eşleştirir */
    background-color: #3498db; /* Arka plan rengini mavi yapar */
    border-radius: 50%; /* Arka planı daire şeklinde yapar */
    z-index: -1; /* Arka planın ikonun arkasında kalmasını sağlar */
    transition: background-color 0.3s ease; /* Hover sırasında arka plan renginin yumuşak geçişle değişmesini sağlar */
}
.social-icon:hover::before {
    background-color: #2980b9; /* Hover sırasında arka plan rengini daha koyu mavi yapar */
}

Sosyal medya ikonlarının arka planını ve hover efektlerini `::before` pseudo-elementi ile oluşturur.

İlerleme Çubuğu

İlerleme çubuğunun doluluk yüzdesini göstermek için `::after` kullanımı:


.progress-bar::after {
    content: "70%"; /* İlerleme çubuğunun doluluk yüzdesini gösterir */
    position: absolute; /* Yüzdeyi çubuğun üstüne yerleştirir */
    right: 10px; /* Yüzdeyi çubuğun sağ tarafına hizalar */
    top: 50%; /* Yüzdeyi çubuğun dikeyde ortasına hizalar */
    transform: translateY(-50%); /* Yüzdeyi tam dikey ortada hizalar */
    color: white; /* Yüzde metnini beyaz yapar */
    font-weight: bold; /* Yüzde metnini kalın yapar */
}

Bu kod, ilerleme çubuğunun doluluk yüzdesini göstermek için `::after` kullanır, bu da kullanıcıya ilerlemenin ne kadar tamamlandığını gösterir.

Bu kod, profil kartının soluna dekoratif bir çizgi ekler, bu da kartın daha dikkat çekici görünmesini sağlar.
